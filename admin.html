🔑 GitHub Token
Token diperlukan untuk upload gambar ke repo GitHub kamu.

Buat di: github.com/settings/tokens
Centang permission: repo (Full control).

Token disimpan hanya di localStorage browser ini.
⚠️ Jangan share token ini ke siapapun. Ini hanya tersimpan di browser kamu.
